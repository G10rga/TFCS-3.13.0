Algorithm Overview
Robinson's Unification (1965):
1. If both terms identical → success
2. If one is a variable → bind (with occurs check)
3. If both are functions → same functor & arity → unify args pairwise
4. Otherwise → fail (clash / arity mismatch)

Occurs Check: ensures variable X is not bound to a term containing X, preventing infinite terms.
